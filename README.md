# Renamer
文件批量重命名程序(基于PyQt5)  
exe文件获取方式在最后  
当前最新版本：`1.1.0`  

**注意！**  
`1.1.X` 版本对 `1.0.X` 版本进行了重构  

### 更新
- 新增：  
  - 新的 GUI界面  
  - 新的 功能  
  - 新的 算法  
- 修复：  
  - 按序命名时后缀不显示
